! Documentação da API Bancária !

informações da API

Aviso: Todos os comandos abaixo requerem a inclusão de uma key como argumento para autenticação.

/create

Cria um usuário no banco de dados bancário. Os seguintes campos são obrigatórios:

  • CPF
  • Nome
  • Email
  • Nome da Mãe (mae)
  • Senha
  • Data de nascimento (dob)

/delete

Deleta um usuário específico. É necessário fornecer o campo id.

/update

Atualiza os dados de um usuário específico. É necessário informar o id e os campos a serem atualizados, como:

  • CPF
  • Nome
  • Email
  • Nome da Mãe
  • Senha

/read

Solicita as informações de um usuário específico. É necessário fornecer o campo id do usuário.
